# Furni - Furniture & Home Decor Website

A modern, responsive furniture and home decor website built with HTML, CSS, and JavaScript.

## 🏠 Project Overview

Furni is a comprehensive furniture and home decor website that showcases beautiful furniture pieces, offers interior design services, and provides an engaging blog for design inspiration.

## 📁 Project Structure

```
furni/
├── index.html              # Home page
├── shop.html               # Shop/Products page
├── about.html              # About Us page
├── services.html           # Services page
├── blog.html               # Blog page
├── contact.html            # Contact page
├── cart.html               # Shopping cart page
├── data.json               # Product data
├── styles/
│   ├── styleHome.css       # Home page styles
│   ├── styleShop.css       # Shop page styles
│   ├── styleAbout.css      # About page styles
│   ├── styleServices.css   # Services page styles
│   ├── styleBlog.css       # Blog page styles
│   ├── styleContact.css    # Contact page styles
│   └── cart.css            # Cart page styles
├── scripts/
│   ├── main.js             # Main JavaScript functionality
│   └── cart.js             # Cart functionality
├── images/                 # Image assets
└── node_modules/           # Dependencies
```

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Local development server (optional)

### Installation
1. Clone or download the project
2. Open `index.html` in your browser
3. Navigate through the pages using the navigation menu

### Development
- All pages are static HTML files
- CSS files are organized by page
- JavaScript functionality is centralized in `scripts/main.js`
- Product data is stored in `data.json`

## 🔧 Customization

### Adding New Pages
1. Create new HTML file following the existing structure
2. Create corresponding CSS file in `styles/`
3. Update navigation links in all pages
4. Add page-specific JavaScript if needed

### Modifying Styles
- Global styles are in each CSS file
- Component-specific styles are grouped
- Responsive breakpoints are clearly defined
- CSS variables can be added for easy theming

### Adding Products
1. Update `data.json` with new product information
2. Ensure product images are in `images/` folder
3. Update shop page to display new products
